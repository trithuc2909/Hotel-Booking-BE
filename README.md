# 🏨 Hotel Management API
API quản lý khách sạn với Node.js, Express, TypeScript và Prisma.

## 📋 Yêu cầu
- Node.js >= 18.x
- PostgreSQL >= 16
- npm

## ⚙️ Cài đặt
1. Clone repository
2. Cài dependencies: `npm install`
<!-- 3. Copy `.env.example` thành [.env](cci:7://file:///d:/PDF-BE/phf-api/.env:0:0-0:0) và điền thông tin database -->
4. Chạy migrations: `npx prisma migrate dev`
5. Chạy server: `npm run dev`

## 🛠️ Tech Stack
- TypeScript
- Express.js
- Prisma ORM
- PostgreSQL
